## Goal
Kick off **ScreenshotFromVideos** — a macOS app that loads a video and exports every Nth frame (or user-selected frames) as PNGs with optional timestamp overlay and chronological numbering. Primary use case: feeding Claude Code visual context from screen recordings of an app in action.

## Decisions

- **Build tooling: xcodegen** (already installed at `/opt/homebrew/bin/xcodegen`) with versioned `project.yml`. Differs from CropBatch / VideoWallpaper (Xcode-managed pbxproj) but gives reproducibility and reviewable diffs. Can be regenerated any time via `xcodegen generate` in `01_Project/`.
- **Conventions matched to siblings**: macOS 15.0+, Swift 6.0, SwiftUI, bundle ID `com.lucesumbrarum.ScreenshotFromVideos`, team `FDMSRXXN73`, notarized non-sandboxed, hardened runtime on.
- **Folder layout** matches CropBatch: `01_Project/`, `02_Design/`, `03_Screenshots/`, `04_Exports/`, `docs/`.
- **Timestamp overlay strategy**: bypass CropBatch's `TextWatermarkVariable` token resolver and pass the formatted timestamp string directly to `applyTextWatermark` per frame — faster than extending the token vocabulary.
